## 🌍 SDG Theme: SDG 3 — Good Health and Well-Being
My course projects revolve around supporting **UN Sustainable Development Goal 3 (SDG 3)**. Specifically, I developed a **Period Tracker Application** designed to empower women by providing a reliable tool to monitor their menstrual health, track physical and emotional symptoms, and easily locate physical healthcare services. This digital solution aims to improve health awareness and overall well-being.

---

## 📱 Project Description & Feature List
The e-Portfolio highlights the progression of building the Period Tracker App through two major milestones:

### 1. Project 1: PeriodTracker Core Features
*   **Core Menstrual Log:** Allows users to easily log period start and end dates.
*   **Symptom & Mood Tracking:** Interface to select and monitor daily physical health data and moods.
*   **Shared ViewModel Architecture:** Smooth communication of dynamic data inputs across different views.

### 2. Project 2: PeriodTracker Advanced Ecosystem
*   **Offline-First Storage:** Local data persistence utilizing a **Room Database** so users can access history without the internet.
*   **Cloud Synchronisation:** Secure, real-time data backup integrated with **Firebase Firestore**.
*   **Intelligent Clinic Locator:** Connects with device **GPS hardware sensors** and fetches public live web endpoints using **Retrofit** to guide users to the nearest clinics.
*   **Material Design 3 (MD3):** Immersive, responsive dark-orchid velvet user interface customized for accessibility.
